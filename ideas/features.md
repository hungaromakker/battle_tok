## Core Gameplay

### Hexagon World System
- Hexagonal tile-based map
- Chain bridges connecting hexagons
- Region capture mechanics
- Flag planting for territory control

### Castle Building
- Fortnite-style building with physics
- Boundary checks for realistic structures
- Main tower as the defensive core
- Multi-level castle construction

### Unit System
- **Workers**: Automated resource gathering and building
- **Actors/Soldiers**: Combat units
- AI-driven distribution - no micromanagement
- Focus on strategic decisions, not unit control

### Resource System
- Minimal starting resources
- Production buildings for projectiles
- Resource management without tedium

### Combat
- Turn-based regional conquest
- Castle sieges
- Projectile-based attacks

## Game Modes

### 1v1 Battle Arena (Priority)
- Single hexagon per player
- Direct castle combat
- Fast-paced matches

### Hex Planet (Future)
- Multiple hexagons
- Strategic conquest
- Longer gameplay sessions

## Technical Features
- Custom Rust engine
- Minimal dependencies
- High performance rendering
- SDF-based graphics (from magic_engine)
